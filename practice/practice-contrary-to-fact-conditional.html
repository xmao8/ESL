Contrary-to-Fact Conditional Practice

Instructions: Complete each sentence using the correct form of the verbs in parentheses to create a contrary-to-fact Conditional statement. Remember to use "were" for the verb "to be" in the 'if' clause.

Name:
Date:
1. If I (have) a million dollars, I (buy) a private jet.
2. If she (be) taller, she (reach) the top shelf.
3. If we (live) in a warmer country, we (go) to the beach every day.
4. If he (know) how to cook, he (make) dinner for us.
5. If they (not have) to work, they (travel) more.
6. If I (be) you, I (accept) that job offer.
7. If it (snow) tomorrow, we (not go) to school.
8. If she (speak) English fluently, she (get) a better job.
9. If I (find) a lost wallet, I (return) it to the owner.
10. If we (have) more time, we (visit) the museum.
11. If he (not be) so shy, he (make) more friends.
12. If I (win) the lottery, I (donate) a lot to charity.
